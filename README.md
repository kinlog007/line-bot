# Line 聊天機器人簡單範例
可參考 [Line 官方文件](https://developers.line.me/bot-api/getting-started-with-bot-api-trial)

## 使用

把 `config.js` 裡面的三個 key 換成自己的

再來先裝必要的套件

```
npm install
```


裝好之後，如果要 debug 的話，就是

```
npm run dev
```

如果想直接跑的，那就

```
npm run build
npm start
```
